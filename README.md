# 프로젝트 주제 

맛집 리뷰 커뮤니티

---
# 프로젝트 수행 방향

   1) 회원가입 / 로그인 기능 구현
   2) API 활용 지도에 맛집 표시
   3) 맛집 검색 기능
   4) 맛집 상세페이지 및 후기 작성 기능
   5) 평점 수렴 후 1,2,3위 표시

---
# 프로젝트 수행 도구

__Back-end__
  + Django
  + MySQL
    
__Front-end__
  + HTML
  + CSS
  + JavaScript
  + Bootstrap
